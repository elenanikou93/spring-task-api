# Spring Task API

Simple Spring Boot REST API.

## Current Features

- Basic Spring Boot setup
- REST endpoint: GET /tasks
- JSON response with list of tasks

- ## To Be Implemented

- Add POST /tasks endpoint (create new task)
- Add GET /tasks/{id} (fetch single task)
- Add DELETE /tasks/{id}
- Add PUT /tasks/{id} (update task)
- Introduce in-memory storage (replace hardcoded list)
- Connect to a database (e.g. H2 or PostgreSQL)

## Endpoints

- GET /tasks → returns a list of tasks

## Tech

- Java 17
- Spring Boot
- Maven
